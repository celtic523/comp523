Interactive Celtic Culture 
Image Upload/Edit Interface

To Install:
	Download the .zip file
	Unzip the package
	Upload the files to a php enabled server
	Create a directory named 'images' in the same directory you uploaded the files to
	
To Use:
	Open a browser and type <yourserver>/imagesindex.html
	Click the 'add image' link to add images
	Once images have been inserted return to the search page to find the inserted images
